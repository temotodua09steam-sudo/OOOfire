# OOOfire - Space Shooter Game

🎮 Интерактивная 2D игра-шутер на Python с использованием Pygame.

## 📋 Требования
- Python 3.8+
- Pygame

## 🚀 Установка

```bash
pip install pygame
```

## 🎮 Как запустить

```bash
python main.py
```

## 🎯 Управление
- **Стрелки/WASD** - движение корабля
- **SPACE** - выстрел
- **ESC** - выход

## 🎨 Особенности
- Враги на космическом фоне
- Система очков
- Взрывы и спецэффекты
- Плавная физика движения
- Уровни сложности
